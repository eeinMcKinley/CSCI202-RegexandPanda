# CSCI202-RegexandPanda
## Basic Program Information
Has a file that takes weather data from the National Weather Service 
and then turns it into a dataframe. From this dataframe, the program turns it into statistics about the weather and boxplots.
Weather Service URL: https://forecast.weather.gov/obslocal.php?warnzone=IAZ031&local_place=Sioux%20City%20IA&zoneid=CDT&offset=18000
